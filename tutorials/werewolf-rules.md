# 狼人杀完整规则与详细教程（12人标准局）

## 1. 基本概念
- **阵营**：好人阵营（村民+神职） vs 狼人阵营。
- **目标**：好人找出并消灭狼人；狼人伪装并屠杀好人。

## 2. 常见身份
- **好人**：预言家（查验）、女巫（解/毒）、守卫（保护）、猎人（带走）、平民。
- **狼人**：普通狼、可自爆。

## 3. 游戏流程
- **夜晚**：闭眼 → 狼人杀人 → 神职依次行动 → 天亮公布死讯。
- **白天**：遗言 → 轮流发言 → 投票放逐最高票者。

## 4. 胜利条件
- 好人：消灭所有狼人。
- 狼人：狼人数量 ≥ 好人数量。

## 5. 进阶技巧
- 发言逻辑、抓刀、站边、反水。
- 房规注意：警长、女巫自救限制等。
- 常见盘面：铁狼、双狼自爆、神民对跳。

**AI助手**：可模拟身份教学、复盘分析具体发言。

**新手建议**：多听逻辑，少冲动投票。
