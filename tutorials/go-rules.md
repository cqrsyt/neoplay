# 围棋（Go）完整规则与详细教程

## 1. 棋具介绍
- **棋盘**：标准19×19路，361个交叉点。初学者推荐9×9或13×13。
- **棋子**：黑白两色，黑棋先行。
- **星位**：9个星位用于定位。

## 2. 基本规则
- 轮流落子于交叉点，落子后不可移动。
- **气**：棋子上下左右相邻空点。无气被提。
- **提子**：对方无气棋子立即移除。
- **禁着点**：禁止自杀落子。
- **打劫**：禁止立即回提，需他处落子。
- **终局**：双方连续虚手（Pass），采用数子法计算胜负（中国规则黑贴3.75子）。

## 3. 死活与基本技巧
- 做眼、破眼、虎口、征子等。
- 布局阶段：占角、守角、挂角。

## 4. 学习建议
- 先练气和死活题。
- 多看定式和名局。
- AI助手可以陪练或讲解具体局面。
