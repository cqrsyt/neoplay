# 国际象棋（Chess）完整规则与教程

## 1. 棋盘与目标
- 8×8格棋盘（64格），白棋先行。
- 目标：将死对方国王（王无法移动且被攻击）。

## 2. 棋子走法（详细）
- **王 (King)**：任意方向1格，不能走到被攻击格。
- **后 (Queen)**：横竖斜任意距离（最强）。
- **车 (Rook)**：横竖任意距离。
- **象 (Bishop)**：斜线任意距离。
- **马 (Knight)**：L形（2+1），可越子。
- **兵 (Pawn)**：直走1格（首步2格），斜吃1格。到底升变（后/车/象/马）。

## 3. 特殊规则
- **将军**：王被攻击，必须立即应将。
- **王车易位**：王和车同时移动（未动过、无阻挡、不被将军）。
- **吃过路兵**：兵特殊吃法。
- **和棋**：无子可动、重复局面、50步无吃子等。

## 4. 策略与学习
- 开局控制中心。
- 中局协调子力，保护王。
- 残局王+兵升变。
- 常用战术：叉子、牵制、发现攻击。
