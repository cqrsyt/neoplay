# 中国象棋完整规则与详细教程

## 1. 棋盘
- 9路×10行，中间有“楚河汉界”。
- 双方各有“九宫”保护将/帅。

## 2. 棋子与走法（红黑对称）
- **帅/将**：在九宫内走1格（横竖），不可出宫，不可对面。
- **士/仕**：斜走1格，只在九宫内。
- **相/象**：走“田”字（2斜），不可过河（塞象眼不能走）。
- **车**：横竖直线任意格（最强攻击力）。
- **马**：走“日”字，先1直再1斜（蹩马腿不能走）。
- **炮**：不吃子时直线走，吃子必须隔一子跳吃。
- **兵/卒**：未过河只向前1格，过河后可向前或横走1格（不能后退）。

## 3. 胜负与规则
- **将死**对方将/帅获胜。
- **相同局面重复**可判和。
- **长将、长捉**等违规判负。

## 4. 策略要点
- 车马炮配合进攻。
- 守好九宫，防对方车炮入侵。
- 中局注重过河兵价值。
- 残局注意士相全 vs 单车等。

**学习建议**：先练基本杀法（如马后炮），多打谱。

**AI助手**：可问具体走法或残局。
